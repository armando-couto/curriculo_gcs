Tutoriais

Instalando ZSH no Ubuntu
apt-get install zsh
apt-get install git-core
wget https://github.com/robbyrussell/oh-my-zsh/raw/master/tools/install.sh -O - | zsh
Escolhendo ZSH como shell padrão da máquina
chsh -s `which zsh`
Reiniciando o computador
sudo shutdown -r 0
Adicionando linhas de ajuste no arquivo .zshenv
echo 'export PATH="$HOME/.rbenv/bin:$PATH"' >> ~/.zshenv
echo 'eval "$(rbenv init -)"' >> ~/.zshenv
echo 'source $HOME/.zshenv' >> ~/.zshrc
exec $SHELL
Clonando o pacote do zsh-autosuggestions depois de instalar o ZSH
git clone https://github.com/zsh-users/zsh-autosuggestions ~/.zsh/zsh-autosuggestions
Rode o comando abaixo para ativar permanentemente o zsh-autosuggestions
echo 'source ~/.zsh/zsh-autosuggestions/zsh-autosuggestions.zsh' >> ~/.zshenv
